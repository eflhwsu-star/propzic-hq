> 최초 생성: 2026-03-27
> 도메인: hq.propai.ai
> 구글 드라이브 위치: 프롭AI/propai-hq

## 프로젝트 개요

PropAI HQ는 34명 AI직원으로 구성된 가상 지휘본부입니다.
집값해독(B2C 부동산 데이터)과 중개오토(B2B 중개사 자동화) 두 서비스를 총괄 관리합니다.

## 배포 구조

| 구성 | 배포 위치 |
|------|-----------|
| index.html (대시보드) | Cloudflare Pages → hq.propai.ai |
| scheduler.py + api_server.py | 가비아 서버 (포트 8001) |
| reports/ | 서버 로컬 저장 |
| logs/ | 서버 로컬 저장 |

## 공통 원칙 (24시간 신속민첩)

- 24시간 365일 즉각 응답
- 답변은 핵심 먼저, 3줄 요약 후 상세 설명
- 문제 발견 시 해결책을 반드시 함께 제시
- 다른 부서 연관사항은 해당 직원 호출 언급
- 긴급도 표시: 🔴긴급 🟡주의 🟢정상
- 항상 한국어로 답변
